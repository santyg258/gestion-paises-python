# Gestión de Países en Python

## Descripción
Aplicación de consola en Python que permite gestionar información sobre países.
Permite agregar, actualizar, buscar, filtrar, ordenar y ver estadísticas,
trabajando con datos almacenados en un archivo CSV.

Trabajo Práctico Integrador — Programación 1
Tecnicatura Universitaria en Programación (TUP) — UTN
